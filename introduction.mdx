Ant Design Pro gives you a fully-featured starting point for enterprise React applications. Built on top of [UMI Max](https://umijs.org/docs/max/introduce) and the [Ant Design](https://ant.design) component library, it ships with role-based access control, internationalization, mock data tooling, a rich set of pre-built dashboard and management pages, and a customizable ProLayout — so you can focus on your business logic from day one.

## Key features

- **TypeScript throughout** — strict typing across all source files with full IDE support
- **UMI Max** — convention-based routing, plugins, and runtime config out of the box
- **Ant Design 5** — comprehensive, enterprise-grade UI component library
- **ProLayout** — configurable sidebar, header, and content layout with dark mode
- **Role-based access control** — protect routes and UI elements with the UMI access plugin
- **OpenAPI code generation** — generate TypeScript service functions from an OpenAPI spec
- **PWA support** — installable progressive web app with service worker
- **Biome linting** — fast, opinionated linting and formatting

<Note>
  Ant Design Pro is a **boilerplate** — you clone it and build your application on top. It is not an npm package or a library you install into an existing project.
</Note>
